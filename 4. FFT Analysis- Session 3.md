# FFT Analysis of Session 3

During the comparative analysis of feature distributions across multiple experimental sessions, **Session 3 exhibited anomalous behavior**. In particular, spectral entropy and kurtosis did not follow the expected trends observed in other sessions. Spectral centroid values were spread out over the same region.

These inconsistencies suggested that the fault characteristics were potentially influenced by **external factors such as structural vibrations, sensor mounting conditions, or environmental noise**. This could imply that a dominant signal due to one of these external factors might be masking the actual signal.

To investigate the underlying cause of these deviations, a **frequency-domain analysis using the Fast Fourier Transform (FFT)** was performed to enable the identification of:

- Additional frequency components  
- Broadband noise  
- Structural resonance effects  

The objective of this analysis was to determine whether the observed feature inconsistencies could be attributed to changes in the spectral characteristics of the vibration signal.

The following sections first highlight the anomalous feature behavior, followed by FFT-based analysis to identify the root cause.
